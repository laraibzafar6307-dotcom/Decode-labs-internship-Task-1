# AI Branding with Image Generation – NovaTech AI

## Overview
This project was created as part of the Generative AI Internship at DecodeLabs. The objective was to design a complete visual identity for a fictional technology startup, **NovaTech AI**, using AI image generation tools and prompt engineering techniques.

The project follows a consistent **Cyberpunk Corporate** branding theme across all generated assets.
